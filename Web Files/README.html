Catalogue Browsing

The original version of Online Bookshop only listed books in the catalogue. It was later extended to support different main categories (books, DVD's and Devices) and sub-categories for every main category.

Payment Processing

At this stage of the system, payment processing is done via SecurePay over SSL. Using SecurePay's test server to handle transactions, the system can also handle rejected transactions delivered from the test server, that is not only due to incorrect credit card information, but also other factors. Namely, responses 34, 43, and 51 are dealt with in the system and provides a summary page related to them; key products to look out for include Galaxy Conflict I, Orange mySlate and The Black Lotus. Response codes 101, 102 and 109 are dealt with by sending the user back to the cart.
